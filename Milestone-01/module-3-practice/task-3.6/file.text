1. fatal: not a git repository – কেন হয়?
উত্তর: আপনি যে ফোল্ডারে git কমান্ড দিচ্ছেন, সেটি Git রিপোজিটরি নয় (অর্থাৎ সেখানে git init করা হয়নি বা .git ফোল্ডার নেই)। সমাধান: git init দিন অথবা সঠিক রিপোজিটরিতে যান।

2. failed to push some refs to remote – কেন হতে পারে?
উত্তর: রিমোট রিপোজিটরিতে আপনার লোকাল ব্রাঞ্চের চেয়ে নতুন কমিট আছে (অন্য কেউ পুশ করেছে)। অথবা আপনার রিমোট URL ভুল। সমাধান: প্রথমে git pull করে কনফ্লিক্ট সমাধান করুন, তারপর git push দিন।

3. Merge conflict কী? কখন সৃষ্টি হয়?
উত্তর: Merge conflict হয় যখন দুটি ভিন্ন ব্রাঞ্চের একই ফাইলের একই লাইনে ভিন্ন পরিবর্তন থাকে। Git বুঝতে পারে না কোনটি রাখবে। উদাহরণ: আপনি和朋友 দুজনে index.html এর ৫ নম্বর লাইনে ভিন্ন কিছু লিখেছেন। তখন merge করার সময় conflict দেখা দেয়।
